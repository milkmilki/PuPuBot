# PuPu Siri 更新日志

## 2026-06-26

### 外观资源插口

- 设置页新增 `外观` tab，可为静息、拖拽、思考、回复、整理记忆、出错和离线状态分别配置图片/动画资源，支持图片/GIF/WebP 以及静音循环视频。
- README 补充外观资源格式清单，明确支持图片、动图、视频、URL/data、本机绝对路径和相对路径。
- 未配置的状态会自动回退到静息资源；静息也未配置时继续使用内置小球。
- 本机绝对路径会通过 Tauri asset protocol 加载，方便直接指向本地 `gif/webp/png/mp4/webm` 等资源文件。
- 修复右键菜单打开后再点 `设置` 时窗口按 collapsed 高度重复计算位移，导致小球整体向上跳的问题。
