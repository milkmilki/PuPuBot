# PuPu Full NPC Assets

Resources for the Content Patcher NPC pack.

- `pupu_sprites.png`: overworld sprite sheet, 16 frames, 16x32 each.
- `pupu_portraits.png`: portrait sheet, 6 frames, 64x64 each, 2 columns x 3 rows.

These are placeholder images. You can replace them with real art later as long
as you keep the same dimensions and filenames, or update `content.json` to point
to new files.
